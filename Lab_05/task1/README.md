# Lab 5.1: Обновление устаревшей версии кода

## Краткое описание

Выполнена переработка старой реализации интернет-магазина с использованием современных возможностей JavaScript (ES6+).

## Структура проекта

- `legacy_code.js` — первоначальная версия приложения
- `legacy_analysis.md` — разбор проблем старого кода
- `modernization_documentation.md` — описание внесённых изменений
- `modernized/` — обновлённая реализация (config.mjs, utils.mjs, core.mjs, main.mjs)

## Основные изменения

| До модернизации | После модернизации |
|-----------------|-------------------|
| `var` | `const` и `let` |
| Конструкторы через `function` | ES6 `class` |
| Открытые свойства объектов | Приватные поля `#` |
| Склейка строк через `+` | Template literals |
| Обычные функции | Стрелочные функции |
| Один глобальный файл | Модульная структура |

## Что улучшено

**Инкапсуляция:** Использование приватных полей (`#items`, `#price`) защищает внутренние данные объектов.

**Модульность:** Логика разделена на отдельные модули (config, utils, core, main), что упрощает поддержку.

**Читаемость:** Современный синтаксис делает код компактнее и понятнее.
