# Документация по обновлению кода (Lab 5.1)

## 1. Разделение на модули

Исходный монолитный файл был разделён на независимые части:

- `config.mjs` — хранение параметров (налог, валюта)
- `utils.mjs` — вспомогательные функции
- `core.mjs` — классы Product и Cart
- `main.mjs` — запуск приложения

Такой подход обеспечивает более чёткую организацию проекта.

---

## 2. Переход на современные стандарты

### Замена `var`
Использованы `const` и `let`, что даёт блочную область видимости и предотвращает случайные ошибки.

### Использование классов
Конструкторы переписаны через `class`, что делает код более структурированным.

### Приватные поля
Добавлены поля с `#`, что защищает внутреннее состояние объектов.

### Template Literals
Строки теперь формируются через `${}` вместо конкатенации.

### Стрелочные функции
Утилитарные функции переписаны в более краткой форме.

- Улучшена читаемость
