# Архитектурная схема: Modular Calculator

## Структура

### Основные модули (static):
- `constants.mjs` — параметры и начальное состояние
- `core.mjs` — математические операции
- `utils.mjs` — логика взаимодействия с интерфейсом
- `app.mjs` — объединяет всё вместе

### Подгружаемые модули (dynamic):
- `advanced_feature.mjs` — расширенные функции
- `lazy_component.mjs` — модуль истории

## Типы импортов

- **Named import** — для арифметических операций
- **Module object import** — для констант
- **Dynamic import** — для отложенной загрузки
