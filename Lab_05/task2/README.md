# Lab 5.2: Модульный калькулятор

## Описание

Реализация калькулятора с использованием ES-модулей.  
В проекте показана работа как со статическими, так и с динамическими импортами.

## Организация модулей

**Модули, загружаемые при старте:**
- `constants.mjs` — базовые настройки и состояние
- `core.mjs` — арифметические вычисления
- `utils.mjs` — обработчики событий и DOM-логика

**Модули, подгружаемые динамически:**
- `advanced_feature.mjs` — расширенные вычисления
- `lazy_component.mjs` — отображение истории операций


## Способы подключения модулей

### Статический импорт

```javascript
import { performCalculation } from './modules/core.mjs';
```

Модуль подключается сразу при загрузке страницы.

### Динамический импорт (по действию пользователя)

```javascript
const module = await import('./modules/advanced_feature.mjs');
```

### Динамический импорт (при появлении блока)
```javascript
import('./modules/lazy_component.mjs');
```
