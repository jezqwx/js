# Технические детали реализации

## Статическая загрузка

Модули `constants`, `core` и `utils` подключаются сразу, так как без них калькулятор не сможет работать.

## Динамическая загрузка

### 1. Advanced функции
Загружаются только при использовании расширенных кнопок.

### 2. История
Модуль истории подключается только когда пользователь прокручивает страницу до соответствующего блока.

## Обработка ошибок

Все динамические импорты защищены `try...catch`.  
При проблемах загрузки выводится сообщение в консоль.
