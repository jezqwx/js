Выбрала utility функцию для проверки email. Она часто используется при регистрации и авторизации пользователей. 

Какую проблему она решает?
Функция решает проблему ввода некоректного email. К примеру без @, без домена и так далее.

В каких контекстах используется?
Используется в регестрации, входа в аккаунт, (валидация, форматирование данных т.д)

Как она решает проблему?
Функция принимает строку и проверяет соответствует ли она формату email с помощью регулярного выражения

Какие ключевые компоненты решения?
входной параметр email, регулярное выражение, логическое значение true / false
